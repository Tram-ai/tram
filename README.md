> ⚠️ **注意**: 本软件目前处于测试阶段，测试完善后将正式开源。

TRAM 是一个在你终端运行的 AI 智能体应用。它可以帮助你理解大型代码库、自动化繁琐的工作，并加快开发速度。

## 安装

确保你已经安装了 Node.js 20 或更高版本。

```bash
npm install -g @tram-ai/tram@latest
```

## 快速开始

安装完成后，在终端运行以下命令开启交互式会话：

```bash
tram
```

初次使用时，按照提示进行配置和认证。你也可以通过 `tram -p "你的问题"` 使用无头模式。
